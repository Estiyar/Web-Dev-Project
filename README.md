## CarHub — Car Marketplace Platform

A web application for buying and selling cars, 
built with Angular and Django REST Framework.

## Project Description

CarHub is a car marketplace where users can browse 
car listings, filter by brand and price, add favorites, 
and leave reviews. Authenticated users can post and 
manage their own car listings.

## Tech Stack
 Frontend: Angular 17
 Backend: Django 
 Auth: JWT
 Database: PostgreSQL
